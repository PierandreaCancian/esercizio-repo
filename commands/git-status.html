GIT cheatsheet

git status

Displays:

  • paths that have differences between the index file and the current HEAD commit
  • paths that have differences between the working tree and the index file
  • and paths in the working tree that are not tracked by Git (and are not ignored by .gitignore)

Options:

  • -s --short: give the output in the short-format.
  • --ignored: show ignored files as well.
